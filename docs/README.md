## 当前架构摘要

```
RK3588 (ELF 2) — 唯一主控
├── I2C → PCA9685 → 12路 PWM → SG90 舵机（柜门锁扣）
├── GPIO → 霍尔传感器（门状态）+ LED（状态灯）
├── UART → PN532 NFC
├── USB → 摄像头 + 条形码扫描器 + 麦克风
├── HDMI → 7寸触摸屏
└── 3.5mm → 扬声器
```
