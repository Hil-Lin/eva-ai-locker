## Phase 3: 硬件控制层 (优先级：高)

> **目标**：能用 Python 代码打开/关闭柜门
> **硬件**：小R科技 PWR.A153 B 扩展板，插入 RK3588 40pin GPIO 排针

### 3.1 PWR.A153 B 硬件接线

| 扩展板接口 | 颜色 | 连接目标 | 说明 |
|-----------|------|---------|------|
| 黑色(GND) | 黑 | 舵机棕色/黑色线 | 必须和 RK3588 共地 |
| 红色(5V) | 红 | 舵机红色线 | 扩展板 5V 稳压供电 |
| 黄色(PWM) | 黄 | 舵机橙色/黄色线 | 经 74HC244 缓冲的 GPIO 信号 |
| DC 插座 | — | 5V/3A 外接电源 | 舵机独立供电 |

**接线口诀**：把舵机 "黑-红-黄" 线和排针顺序对齐，避免接反烧舵机。

### 3.2 GPIO PWM 舵机驱动

**文件**：`src/hardware/__init__.py`, `src/hardware/servo_driver.py`

使用 RK3588 GPIO 硬件 PWM 控制舵机（通过 sysfs PWM 接口或 `lgpio` 库）：

```python
import lgpio  # 或使用 /sys/class/pwm/

class ServoDriver:
    """PWR.A153 B GPIO PWM 舵机驱动"""

    # 8路舵机对应的 GPIO 引脚（按实际接线配置）
    PWM_PINS = [12, 13, 18, 19, 20, 21, 22, 23]  # GPIO编号
    PWM_FREQ = 50       # 舵机标准50Hz
    PWM_PERIOD = 20000  # 周期20ms (单位: μs)

    def __init__(self):
        self.h = lgpio.gpiochip_open(0)  # 打开GPIO芯片

    def set_servo_angle(self, channel, angle):
        """设置舵机角度 (0°-180°)"""
        # 脉宽计算: 0°=500μs, 90°=1500μs, 180°=2500μs
        pulse_width = 500 + (angle / 180.0) * 2000
        duty_cycle = (pulse_width / self.PWM_PERIOD) * 100
        # 通过 lgpio 或 sysfs 设置PWM占空比
        ...

    def open_locker(self, channel):
        """开锁: 舵机转90°"""
        self.set_servo_angle(channel, 90)
        time.sleep(0.8)  # 等待舵机到位
        self.release_locker(channel)  # 停止PWM，防堵转

    def close_locker(self, channel):
        """闭锁: 舵机转0°"""
        self.set_servo_angle(channel, 0)
        time.sleep(0.8)
        self.release_locker(channel)

    def release_locker(self, channel):
        """停止PWM信号 (0%占空比)，防止舵机堵转"""
        ...
```

> **注意**：PWR.A153 B 的 74HC244 芯片只缓冲 PWM 信号（电流<1mA），舵机动力电通过红色 5V 排针由 DC 插座外部电源提供。

```
Phase 1 (数据层) ──→ Phase 2 (AI匹配) ──→ Phase 3 (硬件控制)
                                            │
```

**核心原则**：

3. Phase 1-3 必须最先完成（最小可用原型）
4. Phase 4-6 可并行开发
5. Phase 7-9 在核心功能跑通后进行

## 关键设计决策

| 决策 | 选择 | 理由 |
|------|------|------|
| 架构模式 | RK3588 集中式 | 无 STM32，减少通信延迟和故障点 |
| 舵机控制 | PWR.A153 B 扩展板 + GPIO PWM | 74HC244 缓冲隔离，即插即用，独立5V供电 |
| 柜门锁定 | SG90 舵机锁扣机构 | 比电磁锁更简单、不需12V高压、不需MOSFET |
| 语音识别 | Vosk 离线 | 全程不用网，响应快 |
| 语音合成 | eSpeak (备选 piper-tts) | 轻量离线方案 |
| GUI 框架 | PyQt5 | Buildroot 已有支持，Python 开发快 |
| 数据库 | SQLite | 嵌入式场景最佳选择 |
| 器件数据 | JSON 文件 + SQLite | JSON 便于手动编辑，SQLite 用于运行时查询 |
