# 智能储物柜小型模型组建方案（3柜门原型）

## 1. 项目概述

### 1.1 原型目标
- **柜门数量**：3个（演示级原型）
- **功能演示**：完整用户取物流程（借/还/设置）
- **外观尺寸**：400mm(宽)×300mm(高)×200mm(深)
- **设计风格**：一体式紧凑设计，标准柜体

### 1.2 功能验证清单
1. 主界面显示与触摸交互
2. 人脸识别身份验证
3. NFC校园卡验证
4. AI辅助借用流程
5. 直接借用流程（历史/分类/编号）
6. 条形码扫描归还流程
7. 管理员双重验证与设置功能
8. PCA9685舵机柜门控制与状态监测（RK3588 I2C直连）

### 1.3 架构说明
**本方案采用集中式架构**：RK3588 作为唯一主控，通过 I2C 直连 PCA9685 舵机驱动板控制 12 路 SG90 舵机（柜门锁扣机构），RK3588 GPIO 直接读取霍尔传感器和驱动状态 LED。无中间 MCU，简化系统复杂度。

## 3. 内部组件安装方案

### 3.1 主控模块安装
- **安装位置**：前面板内侧，触摸屏正后方
- **安装方式**：使用铜柱+螺丝固定
- **散热方式**：
  - RK3588芯片加装散热片（20×20×10mm）
  - 散热片上安装小型风扇（25×25×10mm，5V）
  - 风扇通过4pin PWM接口连接RK3588
- **接口方向**：USB/HDMI接口朝下，方便走线
- **电源连接**：通过5V/3A DC-DC模块供电

### 3.2 PCA9685舵机驱动模块安装
- **安装位置**：主控板右侧，使用铜柱固定
- **安装高度**：与主控板同一平面
- **连接方式**：
  - I2C：RK3588的I2C2 ↔ PCA9685（地址0x40）
  - 电源：5V/3A DC-DC模块供电（舵机电源独立）
  - 外部电源：舵机通过PCA9685外部电源端子供电（5V）
- **PWM输出**：16路PWM通道，使用其中3路（可扩展至12路）

### 3.3 电源模块安装
- **安装位置**：机箱底部，远离敏感电路
- **安装方式**：螺丝固定在底板
- **接线要求**：
  - 12V输入线：1.5mm²线径，带保险丝
  - 输出线：0.5mm²线径，不同颜色区分电压
- **绝缘保护**：所有裸露接线点使用热缩管或绝缘胶带

### 3.4 传感器模块安装

#### USB摄像头
- **安装位置**：前面板，触摸屏右上角
- **固定方式**：使用支架+螺丝从内部固定
- **调整角度**：向下倾斜10°，对准用户面部
- **连接方式**：USB延长线连接到RK3588

#### PN532 NFC模块
- **安装位置**：前面板内侧，NFC标识区域背面
- **固定方式**：胶粘或螺丝固定
- **天线方向**：朝向前面板外侧，紧贴面板
- **连接方式**：UART连接到RK3588（或I2C/SPI）

#### 条形码扫描器
- **安装位置**：右侧面，距前面板50mm
- **安装角度**：倾斜15°向上，便于用户扫描
- **固定方式**：支架+螺丝固定
- **窗口设计**：开40×25mm窗口，安装透明亚克力保护片
- **连接方式**：USB延长线连接到RK3588

### 3.5 执行器模块安装

#### 舵机锁扣机构安装
- **安装位置**：每个柜门顶部内侧
- **固定方式**：螺丝固定在柜体上
- **锁扣板**：安装在柜门上，与舵机摇臂对齐
- **驱动方式**：通过PCA9685 PWM控制SG90舵机（0°闭锁/90°开锁）

#### 霍尔传感器安装
- **传感器**：安装在柜体上，与舵机同侧
- **磁铁**：安装在柜门上，与传感器对齐
- **检测距离**：调整至3-5mm有效检测
- **连接方式**：3芯线连接到RK3588 GPIO（带10kΩ上拉电阻）

### 3.6 人机交互模块安装

#### 触摸屏安装
- **安装方式**：从前面板外侧嵌入，内部用支架固定
- **连接线**：
  - HDMI线：连接到RK3588 HDMI输出
  - USB触摸线：连接到RK3588 USB接口
  - 电源：通过RK3588 USB供电或单独5V供电
- **屏幕保护**：出厂自带保护膜

#### 麦克风阵列
- **安装位置**：前面板，触摸屏左上角
- **开孔设计**：3×3mm阵列孔，共6个孔
- **固定方式**：内部支架固定，紧贴面板
- **连接方式**：USB连接到RK3588

#### 扬声器
- **安装位置**：机箱底部，朝前出声
- **开孔设计**：前面板底部开出声孔（4×20mm栅栏孔）
- **固定方式**：螺丝固定
- **连接方式**：3.5mm音频线连接到RK3588

## 4. 电气连接方案

### 4.1 电源分配图
```
AC 220V → 12V/5A开关电源 → 12V总线
    │
    └──→ DC-DC 12V→5V/3A ──→ 5V总线
              │
              ├──→ RK3588开发板 (5V供电)
              ├──→ PCA9685舵机驱动板 (5V供电)
              ├──→ SG90舵机×3 (PCA9685外部电源端子)
              ├──→ 触摸屏 (5V供电)
              ├──→ 条形码扫描器 (USB供电)
              ├──→ 麦克风阵列 (USB供电)
              └──→ DC-DC 5V→3.3V/1A ──→ 3.3V总线
                       │
                       ├──→ NFC模块 (3.3V)
                       ├──→ 霍尔传感器×3 (3.3V+上拉)
                       └──→ 状态LED×3 (3.3V+220Ω电阻，RK3588 GPIO)
```

> **注意**：PCA9685 舵机驱动板需要独立 5V 供电（通过外部电源端子），不可从 RK3588 的 3.3V I2C 引脚取电。舵机堵转电流较大，5V/3A 电源需留足裕量。

### 4.2 信号连接图

#### RK3588 ↔ 外设连接
```
RK3588
├── USB 3.0 Port 1  →  摄像头模块
├── USB 2.0 Port 1  →  触摸屏 (触摸输入)
├── USB 2.0 Port 2  →  麦克风阵列
├── USB 2.0 Port 3  →  条形码扫描器
├── HDMI            →  触摸屏 (显示)
├── UART (GPIO)     →  PN532 NFC模块
├── I2C2 (GPIO)     →  PCA9685舵机驱动板 (地址0x40)
├── GPIO ×3         →  霍尔传感器 (门状态检测)
├── GPIO ×3         →  状态LED (220Ω限流)
└── 3.5mm音频       →  有源音箱
```

#### PCA9685舵机驱动板 ↔ 外设连接
```
PCA9685舵机驱动板
├── I2C (SDA/SCL)   →  RK3588 I2C2
├── VCC (5V)        →  5V电源总线 (外部电源端子)
├── GND             →  电源地
├── PWM通道0        →  SG90舵机1 (柜门1)
├── PWM通道1        →  SG90舵机2 (柜门2)
├── PWM通道2        →  SG90舵机3 (柜门3)
└── PWM通道3-11     →  预留扩展 (柜门4-12)
```

### 4.3 舵机控制电路（PCA9685 + SG90）

#### PCA9685 I2C 连接
```
        RK3588 I2C2
        │
        ├── SDA ──→ PCA9685 SDA (数据线)
        ├── SCL ──→ PCA9685 SCL (时钟线)
        │
        ├── 3.3V ──→ 上拉电阻 4.7kΩ → SDA
        └── 3.3V ──→ 上拉电阻 4.7kΩ → SCL

        PCA9685 电源：
        ├── VCC (5V) ← 5V电源总线
        ├── GND      ← 电源地
        └── V+ (5V)  ← 5V电源总线 (舵机供电端子)
```

#### SG90 舵机连接（以柜门1为例）
```
        PCA9685 PWM通道0 ──→ SG90舵机信号线 (橙色)
        5V电源总线        ──→ SG90舵机电源线 (红色)
        GND               ──→ SG90舵机地线   (棕色)

        舵机角度：
        - 0° (PWM脉宽≈0.5ms)  → 锁扣闭合 (柜门锁定)
        - 90° (PWM脉宽≈1.5ms) → 锁扣打开 (柜门解锁)
        - PCA9685 PWM频率：50Hz (周期20ms)

        注意：每个柜门使用独立的PWM通道
        SG90扭矩：1.5kg·cm，适用于小型锁扣机构
        堵转保护：开锁/闭锁后停止PWM信号，避免长时间堵转

### 4.4 霍尔传感器接口电路
```
        3.3V
         │
         ├──→ 霍尔传感器VCC
         │
         10kΩ上拉电阻
         │
         ├──→ 霍尔传感器OUT ──→ RK3588 GPIO (输入)
         │
         GND ◄── 霍尔传感器GND

         磁铁靠近 → 输出低电平 → 门关闭
         磁铁远离 → 输出高电平 → 门打开
```

### 4.5 状态LED驱动电路
```
        RK3588 GPIO ──→ 220Ω电阻 ──→ LED阳极 ──→ LED阴极 ──→ GND

        GPIO高电平 → LED亮
        GPIO低电平 → LED灭

        颜色定义：
        红色：柜门已借出（锁定）
        绿色：柜门可用（空闲）
        熄灭：系统关机或异常
```
> **注意**：RK3588 GPIO 为 3.3V 电平，可直接驱动 LED（单个 LED 电流约 5mA，在 GPIO 驱动能力范围内），无需额外 MOSFET。

## 5. 组装步骤

### 5.1 组装准备
**工具清单**：
- 螺丝刀套装（十字、一字）
- 万用表（数字万用表）
- 电烙铁套装（焊台、焊锡丝、助焊剂）
- 剥线钳、压线钳
- 热风枪（可选，用于热缩管）
- 尖嘴钳、斜口钳
- 扎带、线卡
- 绝缘胶带、热缩管

**安全注意事项**：
1. 所有组装工作在断电状态下进行
2. 焊接时注意通风，佩戴护目镜
3. 测试前检查所有连接是否正确
4. 首次通电使用限流电源或串入保险丝
5. 注意静电防护（佩戴静电手环）

### 5.2 分步组装流程

#### 第1步：机箱加工（约2天）
1. 根据开孔图加工钣金机箱前面板
2. 在侧面加工条形码扫描器窗口
3. 在背面加工风扇安装孔和散热孔
4. 加工柜门和内部隔板
5. 表面喷塑处理
6. 安装脚垫、铰链等五金件

#### 第2步：电源系统组装（约1天）
1. 安装12V/5A开关电源模块
2. 连接AC输入线（带保险丝座）
3. 安装DC-DC转换模块
4. 连接12V、5V、3.3V总线
5. 安装电源开关和指示灯
6. 测试电源输出：
   - 12V输出：±5%范围内
   - 5V输出：4.75-5.25V
   - 3.3V输出：3.14-3.47V

#### 第3步：主控与舵机控制系统组装（约1天）
1. 安装RK3588开发板（铜柱+螺丝）
2. 安装RK3588散热片和风扇
3. 安装PCA9685舵机驱动板
4. 连接I2C通信线（RK3588 I2C2 ↔ PCA9685）
5. 连接PCA9685电源线（5V独立供电）
6. 连接舵机信号线到PCA9685对应通道
7. 测试通信：
   - `i2cdetect -y 2` 检测PCA9685设备地址(0x40)
   - PWM输出测试（示波器验证50Hz波形）

#### 第4步：传感器安装（约1天）
1. 安装USB摄像头（前面板）
2. 安装PN532 NFC模块（前面板内侧）
3. 安装USB条形码扫描器（侧面）
4. 安装麦克风阵列（前面板）
5. 连接所有传感器到RK3588

### 6.2 PCA9685舵机驱动配置
1. **I2C接口启用**：
   - 在RK3588设备树中启用I2C2
   - 安装 `i2c-tools` 和 `python3-smbus2`
   - 验证设备地址：`i2cdetect -y 2`（应显示0x40）
2. **PCA9685初始化（Python）**：
   - 设置PWM频率为50Hz（舵机标准频率）
   - `set_pwm_freq(50)`
3. **舵机控制接口**：
   - 开锁：`set_pwm(channel, 0, angle_to_pulse(90))` → 90°
   - 闭锁：`set_pwm(channel, 0, angle_to_pulse(0))` → 0°
   - 释放：`set_pwm(channel, 0, 0)` → 停止PWM（省电防堵转）
4. **部署方式**：Python驱动脚本放入 `/opt/smart-locker/hardware/`，通过开机脚本自动加载

## 7. 调试与故障排除

### 7.1 常见问题排查

#### 电源问题
| 现象 | 可能原因 | 解决方法 |
|------|----------|----------|
| 系统无法启动 | 电源未接通或电压不足 | 检查电源指示灯、测量输出电压 |
| RK3588启动后重启 | 供电不足 | 检查5V电源输出电流能力 |
| 舵机转动无力 | 5V供电不足 | 检查PCA9685外部电源，确保5V/3A独立供电 |

#### 通信问题
| 现象 | 可能原因 | 解决方法 |
|------|----------|----------|
| I2C通信失败 | 上拉电阻缺失或地址冲突 | 检查SDA/SCL 4.7kΩ上拉电阻，`i2cdetect` 确认设备地址0x40 |
| USB设备不识别 | 驱动问题或供电不足 | 检查USB供电，确认Linux内核驱动 |
| 触摸屏不响应 | 触摸线未连接或驱动问题 | 检查USB触摸线，查看evtest输出 |

#### 传感器问题
| 现象 | 可能原因 | 解决方法 |
|------|----------|----------|
| NFC读取失败 | 天线方向或距离问题 | 调整天线位置，增加读取距离 |
| 摄像头无图像 | UVC驱动问题 | 检查v4l2设备节点，确认摄像头权限 |
| 条形码无法扫描 | 扫描器位置或角度问题 | 调整扫描器角度，清洁扫描窗口 |

#### 执行器问题
| 现象 | 可能原因 | 解决方法 |
|------|----------|----------|
| 舵机不动作 | I2C通信或PWM配置问题 | `i2cdetect` 检查设备，示波器确认PWM波形 |
| 门状态检测错误 | 霍尔传感器距离问题 | 调整传感器与磁铁对齐位置 |
| LED不亮 | GPIO或电阻问题 | 测量RK3588 GPIO输出电压，检查LED和电阻 |
