# 智能储物柜系统API概览

## 核心模块接口
1. 语音交互模块
   - `VoiceRecognizer`: 语音识别
   - `VoiceSynthesizer`: 语音合成
   - `NLUProcessor`: 自然语言理解

2. AI语义匹配模块
   - `ComponentMatcher`: 器件匹配引擎
   - `RuleEngine`: 规则匹配
   - `SimilarityCalculator`: 相似度计算

3. 身份验证模块
   - `FaceRecognizer`: 人脸识别
   - `NFCReader`: NFC卡读取
   - `AuthManager`: 权限管理

4. 设备控制模块
   - `LockerController`: 柜门控制
   - `SensorMonitor`: 传感器监控
   - `CommunicationManager`: 主从通信
