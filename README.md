# 智能储物柜系统 - 嵌入式芯片与系统设计竞赛项目

基于ELF 2开发板（RK3588）的本地AI智能储物柜系统

## 功能特性
- 本地AI语义匹配：规则引擎+关键词匹配，无需网络
- 语音交互：离线语音识别与合成
- 双重身份验证：人脸识别 + NFC校园卡
- 智能柜门控制：8-12个可扩展柜门
- 全程无网运行：保障实验室数据隐私

## 硬件平台
- 主控：ELF 2开发板（RK3588，6TOPS NPU）
- 从控：STM32F103控制板
- 传感器：USB摄像头、NFC读卡器
- 执行器：12V电磁锁、门状态传感器
- 交互：7寸触摸屏、麦克风、扬声器

## 开发环境
- 主机：Ubuntu 20.04 LTS
- 工具链：ARM交叉编译工具链
- 系统：Buildroot定制Linux
